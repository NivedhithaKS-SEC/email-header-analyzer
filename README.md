# Email Header Analyzer

**Author:** Nivedhitha KS  
**Stack:** Python 3, Flask  
**Purpose:** Detect phishing and email spoofing via header analysis

## What it does
- SPF / DKIM / DMARC authentication checks
- Reply-To mismatch detection
- Display name spoofing detection
- Urgency language analysis
- Mail routing hop analysis
- Risk score 0–100 with verdict

## How to run

```bash
# Step 1 — install Flask
pip install flask

# Step 2 — run the app
python app.py

# Step 3 — open browser
http://127.0.0.1:5000
```
